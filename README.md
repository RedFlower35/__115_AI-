# 班級對抗賽計時系統 (Class Competition Timer System)

![Project Preview](file:///C:/Users/User/.gemini/antigravity/brain/fdceecac-155a-4bee-940b-0550c3b99244/timer_app_preview_1778142868104.png)

> **高精度、雙軌同步、極致美學。**
> 為職能發展學院量身打造的競賽計時解決方案。

---

## 🚀 核心功能 (Core Features)

| 功能 | 描述 |
| :--- | :--- |
| **獨立雙軌追蹤** | 支援 Station 01 與 Station 02 獨立控制，互不干擾。 |
| **高精度邏輯** | 基於 React `useRef` 與 `setInterval` 實現毫秒級穩定計時。 |
| **極致視覺體驗** | 結合 Tailwind CSS 4 與 Framer Motion，提供流暢的微動效與毛玻璃質感。 |
| **直觀操作介面** | 專為觸控與滑鼠操作優化的按鈕設計。 |
